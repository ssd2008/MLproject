# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## [0.3.0] — 2026-07-20

### Technical MVP

- реализован full-stack локальный ассистент для учебных медицинских материалов;
- поддержаны текст, публичные URL, PDF с текстовым слоем и видео;
- добавлена локальная транскрибация видео через `faster-whisper` с тайм-кодами;
- реализованы chunking, dense embeddings, Qdrant retrieval и ответы с цитатами;
- добавлены PostgreSQL-хранилища документов, заданий индексации и feedback;
- подготовлены React frontend, FastAPI API и локальный Docker Compose запуск;
- добавлены unit-, integration- и frontend build-проверки в GitHub Actions;
- закреплены Python- и frontend-зависимости;
- добавлены Apache License 2.0 и Security Policy.

### Retrieval evaluation

На воспроизводимом наборе из 3 материалов, 18 gold-фрагментов и 60 вопросов сравнены BM25, Dense retrieval и Dense + reranker.

| Конфигурация | Recall@5 | MRR | Top-1 gold accuracy | Latency p50 |
|---|---:|---:|---:|---:|
| BM25 | 0.733 | 0.410 | 18.3% | 0.0001 сек. |
| **Dense** | **0.983** | **0.865** | **78.3%** | **0.297 сек.** |
| Dense + reranker | 1.000 | 0.715 | 58.3% | 5.525 сек. |

Dense retrieval на `intfloat/multilingual-e5-large` выбран стандартной конфигурацией. `BAAI/bge-reranker-v2-m3` сохранён как экспериментальная опция и отключён по умолчанию.

### Known limitations

- проект предназначен для локального учебного использования, а не для автономных клинических решений;
- нет authentication, authorization, пользовательской изоляции, rate limiting и resource quotas;
- indexing jobs выполняются внутри FastAPI и не переживают перезапуск;
- PDF-сканы без текстового слоя требуют внешнего OCR;
- локальное файловое хранилище не предназначено для горизонтального масштабирования;
- retrieval benchmark небольшой, англоязычный и не является клинической валидацией;
- пользовательское исследование и итоговые скриншоты будут добавлены отдельно.
